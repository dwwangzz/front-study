• font
  • font-size 浏览器默认字体大小16px 【font-size:16px】
  • font-weight 加粗 bold 等 strong标签
  • font-style 斜体 italic 等 em标签
  • font-family arial字体-乔布斯发明的，现在是互联网公司公用的字体

  • 颜色代码

  • 土鳖式（纯英文单词）red blue yellow
  • 颜色代码 #000000-#ffffff 16进制,如果每两位都相同就可以简写 如：#ff4400 = #f40
  • 颜色函数 rgb(0,0,0) - rgb(255,255,255)
  • 透明色 transparent

  • border[是一个复合属性]

  • border-width 边框线宽度
  • border-style 如：solid
  • border-color
