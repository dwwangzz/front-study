# div
#### 1. div水平居中 设置margin:0 auto兼容各大浏览器让div水平居中

#### 2. div中的元素居中 内部元素增加 margin: auto 属性就可以
如：
```html
<div>
    <table style="margin: auto">
        <tr>
            <td>1</td>
            <td>2</td>
        </tr>
    </table>
</div>
```
